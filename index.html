About Me

Hello! I'm Lagnajit Parida, a versatile professional bridging the worlds of data analytics, engineering, and web development. With a strong foundation in data science and a passion for creating intuitive web applications, I bring a unique blend of analytical thinking and creative problem-solving to every project.

My expertise lies in:

  • Data Analysis and Visualization using Python (Pandas, NumPy, Matplotlib) and SQL
  • Big Data processing with Apache Spark and Hadoop ecosystem
  • Cloud platforms like AWS and Azure for data engineering and analytics
  • Web development using JavaScript, React, and Node.js
  • Machine Learning with scikit-learn and TensorFlow
  • ETL processes and data pipeline design using tools like Apache Airflow

With 8 years of experience in the field, I've honed my skills in both backend data processing and frontend development. This allows me to create end-to-end solutions that not only process data efficiently but also present it in compelling, interactive ways using modern web technologies.

I'm always eager to take on new challenges and contribute to innovative projects that push the boundaries of what's possible with data and web technologies. Whether it's optimizing database queries, building responsive web interfaces, or implementing machine learning models, I'm committed to delivering high-quality solutions that drive business value.
